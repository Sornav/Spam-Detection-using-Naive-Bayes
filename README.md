# Spam-Detection-using-Naive-Bayes
Spam Detection using Multinomial Naive Bayes Classifier,here I have Used Multinomial Naive Bayes Classfier to filter spam SMS using the "spam.csv" which I got from kaggle.I have used Plotly to visualize my data,and for fitting the model i have taken a range of alpha value and chosen the one with the best accuracy and precision,the achieves over 98% accuracy.
